## Overview

**adbcli** is a command-line tool for querying databases via [ADBC (Apache Arrow Database Connectivity)](https://arrow.apache.org/adbc/current/index.html). Built in Rust for high performance and minimal resource usage, adbcli provides both an interactive SQL shell and non-interactive query execution with support for multiple output formats.

## Quick Example

```bash
# Start interactive shell
adbcli --driver duckdb

# Execute a query directly
adbcli --driver duckdb --query "SELECT 42 AS the_answer"

# Export results to JSON
adbcli --driver duckdb --query "SELECT * FROM table" --output result.json
```

## Why ADBC?

ADBC (Apache Arrow Database Connectivity) is a database access API that uses Apache Arrow for data transfer. This provides:

- **Zero-copy data transfer** - Data moves between database and application without serialization overhead
- **Columnar format** - Efficient processing of analytical queries
- **Standardized interface** - Single API works across multiple databases
- **High performance** - Optimized for modern hardware and large datasets
